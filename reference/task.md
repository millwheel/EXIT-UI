# UI 변경 작업

### 헤더
헤더를 없앤다.
로고는 좌측 메뉴 최상단에 위치시킨다.

### 색상

primary color는 그대로 유지한다. #2E7D5A 색상 계열이다.
좌측 메뉴는 reference 이미지에 보이듯이 흰 바탕에 눌렀을 때 primary color로 표시하고 
클릭 된 메뉴는 rounded를 적용한다.

### 프로필, 프로필 수정 버튼, 로그아웃 버튼
프로필, 프로필 수정 버튼, 로그아웃 버튼은 좌측 메뉴 하단에 위치시킨다.

프로필에는 username을 표시하고 
해당 프로필을 누르면 프로필 수정 화면으로 간다.
로그아웃 버튼은 프로필보다 하단에 위치시킨다.

둘다 좌측 메뉴 하단에 위치시킨다.

기본적으로 reference 이미지에 보이는 것처럼 디자인을 변경시킨다.
